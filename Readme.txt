Application reads contacts from the "./data/contacts.json"
All fields are optional and the application will work around the absense of any of them
The "image" field uses images in the "./assets directory, images must be imported in "./assets/images.js" before they can use, otherwise the "./assets/placeholder.jpg" will be used

Run the application using "expo start" in this directory, can be viewed on desktop or mobile by following the instructions presented upon running "expo start"
